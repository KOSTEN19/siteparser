Parsesite: как пользоваться

1) Сначала запустить сервисы

  • Легкий режим: docker compose up -d
  • Полный режим: docker compose --profile full up -d --build
  • Проверка статуса: docker compose ps

3) Что проверять после старта

  • В docker compose ps сервисы должны быть в статусе Up.
  • В папке output/ должны появляться JSON-файлы публикаций.
  • Логи crawler: docker compose logs -f crawler-service
  • Логи scheduler: docker compose logs -f scheduler-service
